# v0.17.1

**Release Date:** 2026-05-18

## Highlights

- **Planetable pyramid-subblock fix** — `get_planetable()` now skips pyramid (reduced-resolution) subblocks when iterating CZI subblocks, so tile indices and row counts are derived exclusively from full-resolution image data.

## Fixed

- `src/czitools/utils/planetable.py`
  - `get_planetable()`: added `if not de.is_pyramid` guard around the per-subblock processing block, preventing pyramid subblocks from creating spurious rows and corrupting tile counters for mosaic/tiled CZI files

## Changed

- `demo/scripts/read_planetable.py`
  - updated `zplane` default from `3` to `0` in the demo `planes` dict to match the sample CZI
- `pyproject.toml` — version bumped to `0.17.1`
- `src/czitools/__init__.py` — `__version__` updated to `0.17.1`

## Added

- `data/testwell96_small.czi` — additional test CZI file covering the fixed planetable code path
